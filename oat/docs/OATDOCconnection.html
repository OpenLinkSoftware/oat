Objects

OAT.Connection(type, optionsObject)

Creates a Connection. Type is one of the type constants (see below). Available options depend on connection type:

  • TYPE_XMLA
    • endpoint
    • dsn
    • user
    • password
  • TYPE_REST
    • url
  • TYPE_WSDL
    • url

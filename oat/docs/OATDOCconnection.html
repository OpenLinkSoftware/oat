Objects

OAT.Connection(type, optionsObject)

Creates a Connection. type is one of the TYPE_constants (see below). Available options depend on connection type :

  • TYPE_XMLA
  • endpoint - ToBeDone
  • dsn - ToBeDone
  • user - ToBeDone
  • password - ToBeDone
  • TYPE_REST
  • url - ToBeDone
  • TYPE_WSDL
  • url - ToBeDone

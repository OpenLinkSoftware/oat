OAT.QuickEdit.assign(element, type, [options])

Marks element as QuickEdit?, i.e. when user clicks it, it transforms into input. Input type is specified via second argument (QuickEdit.STRING => input, QuickEdit.SELECT => select). If QuickEdit.SELECT type is chosen, its options are specified in options.


var text = OAT.Dom.create("span");
text.innerHTML = "hello";
OAT.QuickEdit.assign(text, OAT.QuickEdit.SELECT, ["hello","hi","goodbye"]);

Constants

  • OAT.QuickEdit.STRING - string value
  • OAT.QuickEdit.SELECT - predefined values from the select element

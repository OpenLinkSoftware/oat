Objects

OAT.TagCloud(div, optionsObject)

Creates a Tag Cloud inside div. Options are specified in optionsObject:
  • separator - string, separating individual entries
  • colors - array of colors to be used
  • sizes - array of font sizes to be used
  • colorMapping - one of the COLOR_ constants

Methods

.clearItems()

Removes all items (links).

.addItem(name, link, frequency)

Adds a new item. Its label is name, link is an URL. Third argument, frequency is optional and defaults to 1.

.draw()

Draws the Tag Cloud.
var colors = ["#f00","#0f0","#00f"];
var sizes = ["80%","100%","120%"];
var tc = new OAT.TagCloud("myDiv",{separator:", ", colorMapping:OAT.TagCloudData.COLOR_SIZE, colors:colors, sizes:sizes});

tc.addItem("Label 1", "http://", 10);
tc.addItem("Label 2", "http://", 5);
tc.addItem("Label 3", "http://"); /* frequency == 1 */

tc.draw();

Constants

  • OAT.TagCloudData.COLOR_SIZE - create color based on size (=frequency)
  • OAT.TagCloudData.COLOR_CYCLE - periodically cycle all specified colors
  • OAT.TagCloudData.COLOR_RANDOM - use randomly chosen color for each item

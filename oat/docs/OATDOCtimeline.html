Objects

OAT.Timeline(contentDiv, options)

Creates a Timeline inside contentDiv. options may contain these options:

  • lineHeight - int, height of one record
  • bandHeight - int, height of one band's header
  • margins - int, (default 200)
  • resize - bool (default true)
  • formatter - bool (default true)
  • sliderHeight - int, height of slider
  • autoHeight - bool, should height be automatically adjusted? (default true)

Methods

.addBand(name, color, label)

Adds a data band to chart. The band is identified by name and labeled with label.

.addEvent(bandName, startTime, endTime, content, color)

Adds new event to Timeline, resp. to its band bandName. endTime can be false, when the event has no duration. content is a HTML element; color is a background color for time strip (only when endTime != false).

OAT.Timeline::draw()

Draws the timeline.

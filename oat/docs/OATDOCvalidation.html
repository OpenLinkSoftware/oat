OAT.Validation.create(element, type, options)

Restricts input element. Restriction type is a constant, specified in type. For type see constants section. Options are consisting of:

  • min - minimum number of chars (for TYPE NUMERIC and TYPE_LETTERS) (default 0)
  • max - maximum number of chars (for TYPE NUMERIC and TYPE_LETTERS) (default 50)
  • def - default character (for TYPE NUMERIC and TYPE_LETTERS) (default "_")
  • defDate - default date (default [today.getFullYear(),today.getMonth()+1,today.getDate()])
  • minDate - minimum allowed date (for TYPE_DATE), in form of [year, month, day], all indexed from 1 (default [1900,1,1],)
  • maxDate - maximum allowed date (for TYPE_DATE), in form of [year, month, day], all indexed from 1 (default [2010,12,31])
  • regexp - regular expression (default "")

var i = OAT.Dom.create("input");
OAT.Validation.create(i, OAT.Validation.TYPE_LETTERS, {min:3, max:10, def:"_"}); // 3-10 chars, default '___'

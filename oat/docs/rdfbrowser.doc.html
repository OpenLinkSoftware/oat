OAT.RDFBrowser(div)

Creates a RDF Browser object. Contents will be displayed inside div.

OAT.RDFBrowser::attachData(tripleArray)

Attaches data to the Browser. tripleArray is an array of RDF triples (3-element arrays).

OAT.RDFBrowser::addFilter(predicate, object)

Adds a predicate - object filtering. Any number of such filters can be applied.

OAT.RDFBrowser::removeFilter(predicate, object)

Removes a filter. One must add such filter before removing it.

OAT.RDFBrowser::draw()

Draws the Browser.

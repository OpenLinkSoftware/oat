OAT.Win(paramsObj)

Creates a window. Its main div is accessible as object.dom.container, content as object.dom.content, title as object.dom.title and status area at the bottom as object.dom.status.

The type can contain:

  • OAT.WinData.TYPE_MS - MS Windows classic look
  • OAT.WinData.TYPE_MAC - Mac OS X look
  • OAT.WinData.TYPE_ROUND - rounded corners
  • OAT.WinData.TYPE_RECT - rectangular
  • OAT.WinData.TYPE_ODS - Openlink Data Spaces look
  • OAT.WinData.TYPE_AUTO - guess window look according to browser's platform
  • OAT.WinData.TYPE_TEMPLATE - use own template

Meaning of characters in visibleButtons and enabledButtons from paramsObj:

  • c - show close button
  • m - show minimize button
  • M - show maximize button
  • r - show resize button
  • f - show flip

Argument paramsObj has the following properties:

  • visibleButtons - visible buttons
  • enabledButtons - enabled buttons
  • type - window look
  • template - template information for use with OAT.WinData.TYPE_TEMPLATE
  • className - class for the window
  • x - left coordinate
  • y - top coordinate
  • outerWidth - width
  • outerHeight - height (false means auto)
  • title - window's title
  • status - window's status
